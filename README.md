# ☕ Coffee House

Uma Landing Page moderna e responsiva desenvolvida para a cafeteria Coffee House, criada como um projeto prático para consolidar conceitos de desenvolvimento web e versionamento de código.

## 🛠️ Tecnologias
* **HTML5**: Estruturação semântica da página.
* **CSS3**: Estilização, layout responsivo e design.
* **Git & GitHub**: Controle de versão e hospedagem do código.

## 🚀 Como Executar
1. Faça o clone deste repositório na sua máquina.
2. Abra a pasta do projeto.
3. Clique duas vezes no arquivo `index.html` para abri-lo diretamente em qualquer navegador de internet.
